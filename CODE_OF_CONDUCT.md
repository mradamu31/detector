# Code of Conduct

## Our Standards

FingerTrap is a security-focused open-source project. We expect all contributors and community members to:

- Be direct and respectful — disagree on ideas, not people
- Keep discussions relevant to AI security, detection, and the project
- Give credit where it's due
- Report vulnerabilities responsibly (see [CONTRIBUTING.md](./CONTRIBUTING.md))

## What We Don't Tolerate

- Harassment, personal attacks, or discriminatory language
- Publishing others' private information without consent
- Using this project to harm, surveil, or deceive people
- Deliberately inserting malicious detection bypasses without disclosure

## Scope

This Code of Conduct applies to all project spaces — GitHub issues, PRs, discussions, and any official communication channels.

## Enforcement

Violations can be reported to [security@fingertrap.io](mailto:security@fingertrap.io). We'll review and respond within 72 hours. Repeated or serious violations result in removal from the project.

---

This project adopts a pragmatic, no-drama standard. Build good security tooling. Treat people like adults. Ship.
